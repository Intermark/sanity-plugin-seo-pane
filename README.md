# sanity-plugin-seo-pane

Run Yoast's SEO review tools using Sanity data, inside a List View Pane.

## Installation

```
sanity install seo-pane
```

## Configuration

The plugin can be configured through `<your-studio-folder>/config/seo-pane.json`:

```json
{
  "add-config": "here"
}
```
